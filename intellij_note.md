### 一、命名规范
1、 项目名全部小写  
2、 包名全部小写  
3、 类名首字母大写，如果类名由多个单词组成，每个单词的首字母都要大写。  
如：public class MyFirstClass{}  
4、 变量名、方法名首字母小写，如果名称由多个单词组成，每个单词的首字母都要大写。  
如：int index=0;  
public void toString(){}  
  
5、 常量名全部大写  
如：public static final String GAME_COLOR=”RED”;
